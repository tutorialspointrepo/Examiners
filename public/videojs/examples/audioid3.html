Audio ID3 Tags


ID3 tags plugin allows to read information about audio from ID3 v1 or ID3 v2 tags included in the audio file. This concerns information like audio title, artist, album, release year, genre, album cover, and even song lyrics.

Basic Code Setup


<!--Load player skin, videojs and nuevo plugin inside <body> tag -->
<link href="../skins/nuevo/videojs.min.css" rel="stylesheet" type="text/css" />
<script src="../video.min.js"></script>
<script src="../nuevo.min.js"></script>
<script src="../plugins/videojs.id3.js"></script>

<!--Video tag setup-->
<audio id="player_one" class="video-js vjs-fluid" controls preload playsinline poster="../examples/assets/images/poster.jpg">
	<source src="//opencdn.b-cdn.net/mp3/molly/molly3.mp3" type="audio/mpeg">
</audio>

<!--Initialize the player and the plugin-->
<script>
	var player = videojs('player_one', {license: "key"});
	player.nuevo({ 
		//option_name: option_value, option_name: value
	});
	player.id3();
</script>
You can include a frequency equalizer plugin like in the example above. 
<script>
	var player = videojs('player_one', {license: "key"});
	player.nuevo({ 
		//option_name: option_value, option_name: value
	});
	player.id3();
	player.visualizer({caps:true});
</script>
