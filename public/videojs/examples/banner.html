Overlay Banner on Pause Event

The plugin allows you to display responsive banner ad on every video pause event. The banner can be closed permamently if appropriate closeButton is enabled.
For more options and detailed description check demo on https://www.nuevodevel.com/nuevo/showcase/banner.

Code Setup example (inside <body> tag)

<video id="myplayer" class="video-js vjs-fluid" poster="" controls playsinline poster="../examples/assets/images/coffee.jpg">
	<source src="../examples/assets/video/coffee.mp4" type="video/mp4">
	<source src="../examples/assets/video/coffee.webm" type="video/webm">
</video>

<!--Load CSS and JavaScrips-->
<link href="../skins/treso/videojs.min.css" rel="stylesheet" type="text/css" />
<script src="../video.min.js"></script>
<script src="../nuevo.min.js"></script>

<!--Initialize player and plugins-->
<script>
const player = videojs("myplayer", {license: licenseKey});
player.nuevo({contextMenu: false});
player.banner({
	src: "../examples/assets/images/over.jpg",
	href: "//advertiser-site-url",
	closeButton: true
});
</script>
