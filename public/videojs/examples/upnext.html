UpNext Plugin Example

Unique UpNext plugin by Nuevodevel.com is another addon that might be useful for many webmasters.
It allows to display visual information about next video to play and either play it through the same player or link to the page with next video.

Basic setup

<link href="../skins/treso/videojs.min.css" rel="stylesheet" type="text/css" />
<link href="../plugins/upnext.css" rel="stylesheet" type="text/css" />
<script src="../video.min.js"></script>
<script src="../nuevo.min.js"></script>
<script src="../plugins/upnext.js"></script>

<script>
	var player = videojs('player_1', {license: "key"});
	player.nuevo({ contextMenu:false });
	player.upnext( nextList );
</script>
Check HTML source what is nextList javascript array, it's very similiar to playlist items array. Find more information on plugin's dedicated demo on Nuevodevel.com
