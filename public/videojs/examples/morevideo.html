More videos gallery plugin

Fully responsive, mobile friendly gallery slider of suggested videos.

Code Source

<!--Load player skin, videojs and nuevo plugin inside <body> tag-->
<link href="../skins/nuevo/videojs.min.css" rel="stylesheet" type="text/css" />
<link href="../plugins/morevideo.css" rel="stylesheet" type="text/css" />
<script src="../video.min.js"></script>
<script src="../nuevo.min.js"></script>
<script src="../plugins/morevideo.js"></script>

<!--Video tag setup-->
<video id="player_1" class="video-js vjs-fluid" controls playsinline poster="../examples/assets/images/ski.jpg">
	<source src="//opencdn.b-cdn.net/video/hls/ski/playlist.m3u8" type="application/x-mpegURL">
</video>

<!--Gallery video items array setup-->
<script>
  var more_list = [
	....
  ];
  <!--Initialize the player and plugin-->
  var player = videojs('player_1', {license: "key"});
  player.nuevo();
  player.morevideo(more_list);
</script>
The Gallery plugin can showup on each pause event, exactly the same as on Youtube embed player. Once Gallery is closed manually it does not appear any more. To activate such Gallery layout you need to apply appropriate option.

Code Source

  player.morevideo(more_list, {onpause:true});

Creating an array of videos gallery (more_list) is exactly the same as related videos array. For each video item you need to provide video thumbnail, title, duration and url. You may check what related videos array is in the HTML source code.
