Context Menu on right click

Native context pop-up menu when you right click over player is replaced by custom Nuevo simple menu with 3 basic options (play/pause, mute/unmute and fullscreen) and branding link. You can replace Nuevo branding link with own brand.

Player Code Example


<!--Initialize the player and Nuevo plugin-->
<script>
	const player1 = videojs('player_1', {license: "key"});
	player1.nuevo({ 
		contextUrl: 'https://www.wikipedia.org',
		contextText: 'The Free Ecyclopedia',
		contextIcon: '//www.wikipedia.org/static/favicon/wikipedia.ico',
	}); 
</script>

You can also disable right click context menu at all.

Player Code Setup


<!--Initialize the player and Nuevo plugin-->
<script>
	var player2 = videojs('player_2', {license: "key");
	player2.nuevo({ 
		contextMenu: false, 
		singlePlay:true
	}); 
</script>

Finally you can order to show browsers default menu

Player Code Setup


<!--Initialize the player and Nuevo plugin-->
<script>
	var player3 = videojs('player_3', {license: "key"});
	player3.nuevo({ 
		contextMenu: "default",
		singlePlay:true
	}); 
</script>

Note: Each instance of the player with the nuevo plugin make use of singlePlay enabled option. This allows to play only one player at the same time and pause others automatically.
